# Telegram-бот для онлайн-школы

Интерактивный Telegram-бот, который помогает студентам онлайн-школы получать расписание уроков.  
Бот предоставляет удобный интерфейс с кнопками и прост в использовании.

---

## Цель проекта
Создать Telegram-бота, который:
Отправляет расписание всех уроков по запросу студента;
 Позволяет выбирать класс с помощью кнопок;
 Обеспечивает удобный и понятный интерфейс.


## Инструменты для реализации
- Библиотека pyTelegramBotAPI* 
- SQLite (файл `db.db`)  
- Visual Studio Code  

---

## Функционал
1. Интерактивные кнопки
   - “Показать расписание” — отправляет список всех классов.
   - При выборе класса отображаются предметы по дням недели.
2. Простая структура базы данных
   - Таблица `schedule`: `class`, `day`, `subject`.
3. Интуитивный интерфейс
   - Кнопки для студентов, без сложных команд.
4. Документация
   - Для студентов и администраторов — ниже.

---


---

## Инструкция для администраторов
1. Откройте файл `db.db`.
2. В таблице `schedule` обновите данные:
   - `class` — название класса (например, `5А`, `6Б`).
   - `day` — день недели.
   - `subject` — предмет.
3. Сохраните изменения и перезапустите бота.
