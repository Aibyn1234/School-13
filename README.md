# Проект: Telegram-бот для школы

## Идея проекта
Создать удобного и понятного Telegram-бота для школы, который будет помогать ученикам получать расписание уроков.  
Бот должен быть интерактивным — студенты смогут выбирать свой класс и получать актуальное расписание с помощью кнопок.

---

## Инструменты для реализации 
- Библиотека pyTelegramBotAPI  
- SQLite (для хранения расписания)  
- Visual Studio Code  

---

## Проект состоит из
- config.py — хранит токен бота  
- bot.py — основной код бота  
- db.db — база данных с расписанием уроков  
- requirements.txt — список зависимостей  
- README.md — документация проекта  

---

## Основная цель
Создать Telegram-бота, который помогает ученикам школы узнавать расписание уроков.

## Ключевые возможности
- Кнопка "Показать расписание" — выводит список всех классов  
- Простое и понятное взаимодействие  
- Возможность быстро получить актуальное расписание
